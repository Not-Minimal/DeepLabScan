# Notebooks Directory

This directory contains Jupyter notebooks for data exploration, experimentation, and visualization.

## Suggested Notebooks

- `01_data_exploration.ipynb`: Explore and visualize the dataset
- `02_data_preprocessing.ipynb`: Data cleaning and preprocessing experiments
- `03_model_training.ipynb`: Interactive model training and experimentation
- `04_results_analysis.ipynb`: Analyze training results and metrics
- `05_inference_demo.ipynb`: Demo inference on sample images

## Usage

Use notebooks for:
- Exploratory Data Analysis (EDA)
- Quick prototyping and experimentation
- Visualizing results
- Creating presentations and reports
